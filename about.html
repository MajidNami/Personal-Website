Automatic Vehicle License Plate Recognition System

- Supervised by Dr Ehsan Alirezaei
- Developed a system for automatic recognition of Iranian license plates, applicable in smart parking, traffic control, security, toll collection, and more.
- Created a dataset with vehicle images for license plate recognition and a second dataset for recognizing plate text specifically for Iranian plates.
- Utilized YOLOv8 for license plate detection and both YOLOv8 and Faster R-CNN for plate text recognition.
- Achieved precision metrics: Recall and mAP50 of 0.99 and 0.993 for plate recognition, and for plate text recognition, 0.941 and 0.964 with YOLOv8, and 0.99 and 0.983 with Faster R-CNN on the validation set.
- Developed a user interface using Flask (Python) along with HTML, CSS, and JavaScript, integrating both detection and recognition models into a cohesive system.

Persian Text Processing and Enhancement System

- Course: Software Engineering 1 (Oversaw project as the Head Teaching Assistant, Instructor: Dr Amir Kalbasi)
- Role: Product Manager overseeing project management of 10 teams, each consisting of 4 members.
- Project Summary: Designed and implemented a comprehensive system for Persian text processing that provides a suite of tools to enhance Persian text writing and editing. The system includes features such as real-time autocomplete, which suggests probable next words based on local text data to aid faster typing; spell correction using rule-based methods with the Parsivar and Hazm libraries alongside statistical approaches to improve accuracy; and a synonym and foreign word finder, leveraging datasets and language models (e.g., Word2Vec, GloVe, and FastText) to enrich vocabulary by suggesting synonyms and Persian equivalents for foreign terms. Additionally, the next-word suggestion feature uses n-gram and machine learning algorithms to analyze sentence context and provide relevant word predictions, while the text structure optimization tool refines text layout by adjusting spacing, half-spacing, and punctuation through regex and Hazm library integrations, producing polished and professional output..
- Technologies: Python, JavaScript, SQL, Enterprise Architect, with Parsivar and Hazm libraries, regex, and API integration where needed.
- Managed with Agile methodology and Jira for production management. - Working on the backend and as the production manager in Jira
